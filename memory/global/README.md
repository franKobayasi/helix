# Global Memory

> 跨專案的通用知識。與特定專案或領域無關的原則、工具、流程。

---

## 使用方式

Helix 框架本身的學習和改進記錄。

## 內容結構

```
global/
├── helx-principles/   # Helix 框架的設計原則
├── agent-patterns/     # Agent 協作的模式
├── workflows/         # 開發流程的改進
└── README.md          # 本檔案
```

## 維護原則

- 全域記憶影響所有專案，修改要謹慎
- 當框架本身有重大改進時，更新這裡
- 確保任何人都能從這裡了解 Helix 的核心理念

## 與其他層級的區別

| 層級 | 內容 | 壽命 |
|------|------|------|
| project/ | 單一專案的知識 | 專案期間 |
| domain/ | 主題領域的模式 | 長期 |
| global/ | 框架本身的原則 | 永久 |
