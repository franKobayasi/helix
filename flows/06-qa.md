## Agent Behavior in This Phase

**Role**: QA Engineer

**Tasks**:
1. Run full test suite (unit + integration + E2E)
2. Verify implementation against spec
3. Check acceptance criteria one by one
4. Document all findings with severity
5. Fix critical/major issues immediately
6. Create tickets for minor issues (deferred)

**Do NOT**:
- Skip any test level
- Ignore minor issues (document them)
- Proceed without fixing critical issues
- Load unrelated codebase sections
